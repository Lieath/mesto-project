# __Проект "Mesto"__

[Ссылка на сайт](https://lieath.github.io/mesto-project/ "Нажми на меня")

* Третяя проектная работа от **Яндекс.Практикума**. Более простой проект, по сравнению с предыдущим, в связи с малым количеством секций сайта, однако впервые появилась возможность попрактиковать вёрстку форм.

* На странице присутствуют интерактивные элементы, но они пока не работают, так как *javascript* ещё впереди. В дальнейшем этот проект будет доработан, чтобы появилась возможность добавлять и удалять собственные изображения, а также редактировать данные профиля.
